# Code of Conduct (short)

We want this project to be welcoming, respectful, and collaborative.

Expected behavior
- Be kind and constructive in code reviews and issues.
- Assume good intent and help newcomers with patience.

Unacceptable behavior
- Harassment, hate speech, or targeted attacks will not be tolerated.

Reporting
- If you encounter problems, open a private issue or email the maintainers.

Thank you for helping keep this community friendly.
