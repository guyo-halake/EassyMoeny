Onboarding checklist for new contributors

- [ ] Read `CONTRIBUTING.md` and follow commit conventions
- [ ] Pick a small UI or copy task (pref: microcopy, accessibility)

Accessibility checklist (for screens)

- [ ] Color contrast is sufficient

Commit & release guidance
